

*  *screenshot tool for the presentation template*

Use ```chromium``` through ```selenium``` & ```phantomjs```

## Installation

    $ make install

## Usage  
        
### start geckodriver   
   
    $ ./geckodriver

### capture

    # url pointing to presentation template
    # ex: http://padagraph.io/presentation/strains
    $ python screenshot.py url


Open image ```crop.png```
